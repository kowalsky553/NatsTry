1. Развернуть базу данных из PostTechTest.bak;
2. Запустить influxd.exe

====================================
Пояснения к работе:
Не совсем была понятна формулировка, из которой надо было понять, как создавать сообщения, которые хранятся в базе,
поэтому сделал цикличную загрузку из БД. При потере соединения с сервером Издатель останавливается, пока не восстановит
соединение с nats.

Подписчик получает сообщение и записывает в БД Influx, при необходимости, создает новую БД с именем, заданным в App.config.
